Tentang Saya

Saya adalah UI/UX Designer berbasis di Sumbersari-Kesamben, dengan passion mendalam terhadap teknologi dan desain. Dengan pengalaman 14 tahun di industri digital, saya berkomitmen menciptakan solusi inovatif yang fungsional, estetis, dan user-friendly. Di luar pekerjaan, saya suka membaca buku teknologi dan menjelajah alam untuk inspirasi.

  • Web Developer di PT. Tech Solutions (2020 - Sekarang): Mengembangkan situs responsif dengan HTML, CSS, JS, dan React. Meningkatkan performa 30% via optimasi kode dan API.
  • Freelance Web Designer (2018 - 2020): Mendesain ulang e-commerce dan app mobile menggunakan Figma/Adobe XD. Kompatibilitas lintas perangkat dan ulasan positif dari klien.
  • Pendidikan: Sarjana Teknik Informatika, Universitas Indonesia (2014 - 2018): Lulus cum laude, fokus pengembangan software. Proyek akhir: Aplikasi web AI yang dikembangkan jadi prototipe.
  • Proyek Pribadi: Aplikasi To-Do List Pintar (2022): App web dengan AI untuk prioritas tugas. Dibangun dengan Node.js/MongoDB, diakses 500+ pengguna.
